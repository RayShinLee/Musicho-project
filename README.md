# Musicho-project
NCCU Information Systems Project 2019

針對所有喜歡音樂與分享音樂的大眾，創造可以使用音樂為主題分享心情與生活的平台，歌手或樂團與粉絲也可以在這個平台進行互動，宣傳推廣，特別是對於原創的歌手與獨立樂團。初期發展而言，獨立樂團更需要也可以透過這個平台增加聽眾人數，使用者可以以音樂為主題接收到關於朋友分享的內容，也可以讓使用者建立自己的風格，推薦與分享喜歡的歌曲給朋友與大眾。
